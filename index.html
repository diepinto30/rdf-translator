Please note that the service utilizes RDFLib 3.1.0 to parse and serialize between data formats.

Rest API

This service provides an API which allows for two possibilities:

  • "url", "if", and "of"
  • "url" and "of", while "if" is determined automatically

The underlying pattern is (where the usage of "if" is more restrictive and optional):
http://rdf-translator.appspot.com/parse?url=<page_url>&if=<input_format>&of=<output_format>
